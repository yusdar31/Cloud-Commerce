# 1. Purpose

Dokumen ini berisi referensi visual dan UX yang menjadi inspirasi dalam membangun CloudCommerce.

Dokumen ini **bukan** digunakan untuk menyalin desain.

Tujuannya adalah memahami:

- UX Pattern
- Interaction
- Motion
- Layout
- Typography
- Navigation
- Visual Hierarchy
- Component Behaviour

CloudCommerce akan mengombinasikan berbagai inspirasi menjadi identitas visual yang unik.

# 22. Design Review Checklist

Sebelum halaman dianggap selesai.

Apakah:

✓ Tidak terlihat seperti template?

✓ Tidak terlihat AI?

✓ Konsisten?

✓ Responsif?

✓ Accessible?

✓ Motion secukupnya?

✓ CTA jelas?

✓ Typography baik?

✓ Navigation sederhana?

Jika salah satu tidak terpenuhi.

Halaman harus direvisi.
